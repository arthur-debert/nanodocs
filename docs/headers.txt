Headers separate files in the bundled document. By default, they show the filename.

OPTIONS:
--no-header             Hide headers completely
--header-seq=TYPE       Add sequence numbers (numerical, letter, roman)
--header-style=STYLE    Change format (filename, path, nice)

EXAMPLES:
"file.txt"              Default style (filename)
"/path/to/file.txt"     Path style (full path)
"File (file.txt)"       Nice style (titled with original in parentheses)
"1. file.txt"           With numerical sequence
"a. File (file.txt)"    Combined letter sequence with nice style
